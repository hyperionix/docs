hp.logger
logger module to create loggers (see usage below)

Usage:

    local LOG = hp.logger:new()
LOG:dbg("debug")
LOG:wrn("warning")
LOG:err("error")
LOG:pp({a = 5}) -- print tables
LOG:pp({a = 5}, 1) -- print tables with depth
