Hook

Usage:

Hook {
 name = "MyFunctionHook",
 target = "module!MyFunctionHook",
 decl = [[ int MyFunction(int param1, int param2); ]],
 convention = "stdcall",
 onEntry = function(context)
 end,
 onExit = function(context)
 end
}
Hook.name
Hook name. The hook name should be unique across all packages repositories. TODO: Implement packages namespaces
Hook.target
Hook target. Could be modname!Function or modname!Offset.
Hook.decl
Target function prototype.
Hook.convention

[opt] target function calling convention. Could be:

  • "fastcall" - default for 64 bit apps
  • "stdcall" - default for 32 bit apps
  • "thiscall"
Hook.onEntry
[opt] HookCallbacks.onEntry callback.
Hook.onExit
[opt] HookCallbacks.onExit callback.
