Tables

Probe.HookEntry
Hook entry for the probe declaration.

At least onEntry or onExit must be defined

Fields:

  • name Name of the hook. The hook should exists in the accessible list of packages.
  • onEntry [opt] HookCallbacks.onEntry callback.
  • onExit [opt] HookCallbacks.onExit callback.
  • onBlock [opt] HookCallbacks.onBlock callback.

Probe

Usage:

Probe {
 name = "My Probe",
 hooks = {
   {
     name = "MyFunctionHook",
     onEntry = function(context)
     end,
     onExit = function(context)
     end,
     onBlock = function(context)
     end
   }
 }
}
