Functions

share.add (shareName, sharePath[, shareType=STYPE_DISKTREE[, serverName]])
Add share

Parameters:

  • shareName share name
  • sharePath share path
  • shareType share type (default STYPE_DISKTREE)
  • serverName server DNS or NetBIOS name (if not specified, the local computer is used) (optional)

Returns:

    0 on success
share.check (sharePath, shareType[, serverName])
Check whether or not a server is sharing a resource with specified path

Parameters:

  • sharePath path to check for shared access
  • shareType LPDWORD that on success receives a bitmask of flags that specify the type of the shared resource
  • serverName server DNS or NetBIOS name (if not specified, the local computer is used) (optional)

Returns:

    0 on success
share.delete (shareName[, serverName])
Delete share

Parameters:

  • shareName share name
  • serverName server DNS or NetBIOS name (if not specified, the local computer is used) (optional)

Returns:

    0 on success
share.forEach (func[, level=0[, serverName]])
Enumerate shares and call the function for each

Parameters:

  • func function to be called for each share
  • level information level of the data (default 0)
  • serverName server DNS or NetBIOS name (if not specified, the local computer is used) (optional)
