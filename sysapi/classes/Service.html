Service.create (scmHandle, name, binPath[, access=SERVICE_ALL_ACCESS[, serviceType=SERVICE_WIN32_OWN_PROCESS[, startType=SERVICE_DEMAND_START[, errControl=SERVICE_ERROR_NORMAL[, account[, password]]]]]])
Create a service The function shouldn't be used directly, use ServiceManager:createService instead

Parameters:

  • scmHandle handle to SCM
  • name name of the service
  • binPath fully qualified path to the service binary file, can also include arguments
  • access desired access to the service (default SERVICE_ALL_ACCESS)
  • serviceType service type (default SERVICE_WIN32_OWN_PROCESS)
  • startType service start options (default SERVICE_DEMAND_START)
  • errControl severity of the error, and action taken, if this service fails to start (default SERVICE_ERROR_NORMAL)
  • account name of the account under which the service should run (optional)
  • password password to the account name specified by the account parameter (optional)

Returns:

    Service object
